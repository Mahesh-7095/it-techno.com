HTML Input Attributes

Common Input Attributes

HTML input elements support attributes like placeholder, required, readonly, and disabled.


<form>
    <label for="placeholder">Placeholder:</label>
    <input type="text" id="placeholder" name="placeholder" placeholder="Enter text">

    <label for="required">Required:</label>
    <input type="text" id="required" name="required" required>

    <label for="readonly">Readonly:</label>
    <input type="text" id="readonly" name="readonly" value="Readonly" readonly>

    <label for="disabled">Disabled:</label>
    <input type="text" id="disabled" name="disabled" value="Disabled" disabled>

    <button type="submit">Submit</button>
</form>
