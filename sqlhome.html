Welcome to SQL Home

This page demonstrates basic SQL concepts with examples and code snippets.

1. SELECT Query

Retrieve all records from a table.


SELECT * FROM employees;

Example Table: Employees

ID Name Department Salary
1 John Doe HR 50000
2 Jane Smith IT 60000
3 Alice Johnson Finance 55000

Output: Displays all rows from the employees table.

2. INSERT Query

Add a new record to a table.


INSERT INTO employees (ID, Name, Department, Salary)
VALUES (4, 'Bob Brown', 'Marketing', 48000);

Result: A new row is added to the employees table.

3. UPDATE Query

Modify an existing record in a table.


UPDATE employees
SET Salary = 52000
WHERE ID = 1;

Result: The salary of the employee with ID = 1 is updated to 52000.

4. DELETE Query

Remove a record from a table.


DELETE FROM employees
WHERE ID = 3;

Result: The employee with ID = 3 is removed from the employees table.

5. WHERE Clause

Filter records based on a condition.


SELECT * FROM employees
WHERE Department = 'IT';

Output: Displays only employees in the IT department.

6. ORDER BY Clause

Sort records in ascending or descending order.


SELECT * FROM employees
ORDER BY Salary DESC;

Output: Displays employees sorted by salary in descending order.
