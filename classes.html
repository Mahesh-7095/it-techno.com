HTML Classes

Using CSS Classes

CSS classes allow you to apply styles to multiple elements. Use the class attribute to assign a class to an element. 


<p class="highlight">This is a highlighted paragraph.</p>
<p class="warning">This is a warning paragraph.</p>

This is a highlighted paragraph.

This is a warning paragraph.

Multiple Classes

You can assign multiple classes to an element by separating them with spaces. 


<p class="highlight warning">This paragraph has multiple classes.</p>

This paragraph has multiple classes.
