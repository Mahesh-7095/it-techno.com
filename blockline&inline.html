HTML Block & Inline Elements

Block-Level Elements

Block-level elements start on a new line and take up the full width of their container. Examples include <div>, <p>, and <h1>. 


<div class="block">This is a block-level element.</div>
This is a block-level element.

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples include <span>, <a>, and <strong>. 


<span class="inline">This is an inline element.</span>
This is an inline element.
