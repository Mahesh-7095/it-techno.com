HTML Input Form Attributes

Form-Specific Input Attributes

Input elements can have form-specific attributes like form, formaction, formenctype, and formtarget. 


<form id="form1">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">

    <button type="submit" formaction="/submit1" formtarget="_blank">Submit 1</button>
    <button type="submit" formaction="/submit2" formtarget="_self">Submit 2</button>
</form>
