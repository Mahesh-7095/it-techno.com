Welcome to Scripting!

Scripting involves writing small programs (scripts) to automate tasks or manipulate data. Scripts are typically written in languages like Python, Bash, or PowerShell and are used to simplify repetitive tasks, manage systems, and process data.

What is Scripting?

Scripting is the process of writing code to automate tasks that would otherwise be performed manually. Scripts are often used for system administration, data processing, and task automation.

Why Learn Scripting?

  • Automation: Save time by automating repetitive tasks.
  • Efficiency: Perform complex tasks with minimal effort.
  • Flexibility: Use scripting languages to solve a wide range of problems.

import os

# Rename files in a directory
directory = "path/to/directory"
for filename in os.listdir(directory):
    if filename.endswith(".txt"):
        new_name = filename.replace("old_", "new_")
        os.rename(os.path.join(directory, filename), os.path.join(directory, new_name))
        print(f"Renamed {filename} to {new_name}")
