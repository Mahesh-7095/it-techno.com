HTML Colors

Introduction to Colors in HTML

Colors in HTML and CSS can be defined in several ways, including:

  • Color Names: Predefined color names like red, blue, green, etc.
  • RGB Values: Using the rgb() function to define colors by their red, green, and blue components.
  • Hexadecimal Values: Using a 6-digit hexadecimal code like #FF5733.
  • HSL Values: Using the hsl() function to define colors by their hue, saturation, and lightness.

Below are examples of each method:

Color Names

HTML supports 140 predefined color names. Here are a few examples: 


<div style="background-color: red;"></div>
<div style="background-color: blue;"></div>
<div style="background-color: green;"></div>

RGB Colors

RGB colors are defined using the rgb() function. Each parameter (red, green, blue) ranges from 0 to 255. 


<div style="background-color: rgb(255, 0, 0);"></div>
<div style="background-color: rgb(0, 255, 0);"></div>
<div style="background-color: rgb(0, 0, 255);"></div>

Hexadecimal Colors

Hexadecimal colors are defined using a 6-digit code prefixed with a #. Each pair of digits represents red, green, and blue. 


<div style="background-color: #FF5733;"></div>
<div style="background-color: #33FF57;"></div>
<div style="background-color: #3357FF;"></div>

HSL Colors

HSL colors are defined using the hsl() function. The parameters are hue (0-360), saturation (0%-100%), and lightness (0%-100%). 


<div style="background-color: hsl(0, 100%, 50%);"></div>
<div style="background-color: hsl(120, 100%, 50%);"></div>
<div style="background-color: hsl(240, 100%, 50%);"></div>
