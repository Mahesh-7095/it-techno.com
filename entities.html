HTML Entities

Common HTML Entities

HTML entities are used to display special characters that are not available on the keyboard or have special meanings in HTML. 


<p>Less than: &lt;</p>
<p>Greater than: &gt;</p>
<p>Ampersand: &amp;</p>
<p>Copyright: &copy;</p>
<p>Registered Trademark: &reg;</p>

Greater than: >
