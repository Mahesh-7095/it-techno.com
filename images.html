HTML Images

Embedding Images

Use the <img> tag to embed images in HTML. The src attribute specifies the image path, and the alt attribute provides alternative text. 


<img src="example.jpg" alt="Example Image">

Image with Width and Height

You can specify the width and height of an image using the width and height attributes. 


<img src="example.jpg" alt="Example Image" width="300" height="200">

Image as a Link

You can wrap an image in an <a> tag to make it a clickable link. 


<a href="https://www.example.com">
    <img src="example.jpg" alt="Example Image">
</a>
