HTML Table Styling

Styling Tables with CSS

Use CSS to style tables, including alternating row colors and hover effects. 


<style>
    th {
        background-color: #007BFF;
        color: #fff;
    }
    tr:nth-child(even) {
        background-color: #f9f9f9;
    }
    tr:hover {
        background-color: #f1f1f1;
    }
</style>
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>25</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>30</td>
        <td>Los Angeles</td>
    </tr>
</table>
