HTML Attributes Examples

href Attribute

The href attribute specifies the URL of the page the link goes to. 


<a href="https://www.example.com">Visit Example.com</a>

src Attribute

The src attribute specifies the path to the image to be displayed. 


<img src="image.jpg" alt="Description of image">

alt Attribute

The alt attribute provides alternative text for an image if it cannot be displayed. 


<img src="image.jpg" alt="Description of image">

style Attribute

The style attribute is used to add inline styles to an element. 


<p style="color: red;">This is a red paragraph.</p>

class Attribute

The class attribute is used to specify one or more class names for an element. 


<p class="highlight">This paragraph has a class of 'highlight'.</p>

id Attribute

The id attribute specifies a unique id for an element. 


<p id="unique">This paragraph has a unique id.</p>

title Attribute

The title attribute provides additional information about an element, often shown as a tooltip. 


<p title="This is a tooltip">Hover over this paragraph.</p>

disabled Attribute

The disabled attribute disables an input element.


<input type="text" disabled placeholder="This input is disabled">

readonly Attribute

The readonly attribute makes an input element read-only. 


<input type="text" readonly value="This input is read-only">

required Attribute

The required attribute specifies that an input field must be filled out before submitting the form. 


<input type="text" required placeholder="This field is required">
