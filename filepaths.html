HTML File Paths

Relative File Paths

Relative file paths are used to link to files within the same website. They are relative to the current file. 


<img src="images/example.jpg" alt="Example Image">

Example: images/example.jpg refers to a file in the images folder.

Absolute File Paths

Absolute file paths include the full URL to the file.


<img src="https://www.example.com/images/example.jpg" alt="Example Image">

Example: https://www.example.com/images/example.jpg refers to a file on an external server.
