HTML Text Formatting

Bold Text

The <b> tag is used for bold text.


<b>This text is bold.</b>

Italic Text

The <i> tag is used for italic text.


<i>This text is italic.</i>

Underlined Text

The <u> tag is used for underlined text. 


<u>This text is underlined.</u>

Strong Text

The <strong> tag is used for strong importance (typically displayed as bold). 


<strong>This text is strong.</strong>

Emphasized Text

The <em> tag is used for emphasized text (typically displayed as italic). 


<em>This text is emphasized.</em>

Strikethrough Text

The <s> tag is used for strikethrough text. 


<s>This text is strikethrough.</s>
