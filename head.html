HTML Head Element

Understanding the <head> Element

The <head> element contains metadata, links to stylesheets, scripts, and other resources. It does not display content directly. 


<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
    <meta name="description" content="Page description">
    <meta name="keywords" content="HTML, CSS, JavaScript">
    <meta name="author" content="Author Name">
    <link rel="stylesheet" href="styles.css">
</head>

Common Elements in <head>

  • <title>: Defines the title of the document (shown in the browser tab).
  • <meta>: Provides metadata like character set, viewport, and descriptions.
  • <link>: Links to external resources like stylesheets.
  • <style>: Contains internal CSS styles.
  • <script>: Links to or contains JavaScript code.
