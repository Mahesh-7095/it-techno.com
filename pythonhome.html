Welcome to Python Home

This page demonstrates basic Python concepts with examples and code snippets.

1. Variables and Data Types

Python supports various data types like integers, floats, strings, and booleans. 


# Variables
name = "Alice"
age = 25
height = 5.6
is_student = True

# Output
print("Name:", name)
print("Age:", age)
print("Height:", height)
print("Is Student:", is_student)

2. Loops

Python supports for and while loops for iteration. 


# For Loop
for i in range(5):
    print("Iteration:", i)

# While Loop
count = 0
while count < 3:
    print("Count:", count)
    count += 1

3. Functions

Functions are defined using the def keyword.


# Function Definition
def greet(name):
    return f"Hello, {name}!"

# Function Call
message = greet("Bob")
print(message)

4. File Handling

Python can read from and write to files.


# Writing to a File
with open("example.txt", "w") as file:
    file.write("Hello, Python!")

# Reading from a File
with open("example.txt", "r") as file:
    content = file.read()
    print("File Content:", content)

5. Lists and Dictionaries

Lists and dictionaries are commonly used data structures in Python. 


# List
fruits = ["apple", "banana", "cherry"]
print("Fruits:", fruits)

# Dictionary
person = {"name": "Alice", "age": 25, "city": "New York"}
print("Person:", person)
