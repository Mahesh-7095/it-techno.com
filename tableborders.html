HTML Table Borders

Styling Table Borders

Use CSS to style table borders. The border property can be applied to <th> and <td> elements. 


<style>
    th, td {
        border: 2px solid #007BFF;
    }
</style>
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>25</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>30</td>
        <td>Los Angeles</td>
    </tr>
</table>
