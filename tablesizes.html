HTML Table Sizes

Controlling Table Sizes

Use the width property to control the size of a table. 


<style>
    table {
        width: 50%;
    }
</style>
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>25</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>30</td>
        <td>Los Angeles</td>
    </tr>
</table>
