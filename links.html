HTML Links

Basic Hyperlink

Use the <a> tag to create a hyperlink. The href attribute specifies the destination URL. 


<a href="https://www.example.com">Visit Example.com</a>

Visit Example.com

Link to Another Page

You can link to another page on your website by specifying the relative path in the href attribute. 


<a href="about.html">About Us</a>

About Us

Link to an Email Address

Use the mailto: protocol to create a link that opens the user's email client. 


<a href="mailto:example@example.com">Send Email</a>

Send Email

Link to a Phone Number

Use the tel: protocol to create a link that dials a phone number on mobile devices. 


<a href="tel:+1234567890">Call Us</a>

Call Us
