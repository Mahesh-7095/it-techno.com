Description Lists

Use the <dl> tag to create a description list. Each term is defined with <dt>, and its description is defined with <dd>. 


<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
    <dt>JS</dt>
    <dd>JavaScript</dd>
</dl>
