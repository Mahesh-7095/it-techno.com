HTML ID Attribute

Using the ID Attribute

The id attribute is used to uniquely identify an element. Each id must be unique within a page. 


<p id="unique">This is a uniquely styled paragraph.</p>

This is a uniquely styled paragraph.

ID vs Class

While class can be used on multiple elements, id should be unique to a single element. 


<p id="unique">This is a unique element.</p>
<p class="highlight">This is a class-based element.</p>

This is a unique element.

This is a class-based element.
