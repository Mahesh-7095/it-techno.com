Displaying Code

Use the <code> tag to display inline code snippets. 


<p>Use the <code>&lt;code&gt;</code> tag to display code.</p>

Preformatted Text

Use the <pre> tag to display preformatted text, such as code blocks. 


<pre>
    <code>
        function greet() {
            console.log("Hello, World!");
        }
    </code>
</pre>

Keyboard Input

Use the <kbd> tag to represent keyboard input.
