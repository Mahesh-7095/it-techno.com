HTML Div Element

Using the <div> Element

The <div> element is a block-level container used to group and style content. 


<div class="box">
    This is a div element.
</div>
This is a div element.

Grouping Content with Div

You can use multiple <div> elements to group and style different sections of content. 


<div class="box">
    <h3>Section 1</h3>
    <p>This is the first section.</p>
</div>
<div class="box">
    <h3>Section 2</h3>
    <p>This is the second section.</p>
</div>
