HTML Character Sets

Specifying Character Sets

The <meta charset="..."> tag is used to specify the character encoding for the HTML document. 


<meta charset="UTF-8">

UTF-8 is the most commonly used character encoding and supports all Unicode characters.

Common Character Sets

  • UTF-8: Supports all Unicode characters (recommended).
  • ISO-8859-1: Supports Western European characters.
  • Windows-1252: Similar to ISO-8859-1 but with additional characters.
