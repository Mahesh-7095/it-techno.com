HTML vs XHTML

Differences Between HTML and XHTML

HTML and XHTML are both markup languages, but they have some key differences:

  • Syntax: XHTML is stricter and requires well-formed XML syntax.
  • Case Sensitivity: XHTML is case-sensitive, while HTML is not.
  • Self-Closing Tags: XHTML requires self-closing tags (e.g., <br />).
  • DOCTYPE: XHTML uses a different DOCTYPE declaration.

HTML Example


<!DOCTYPE html>
<html>
<head>
    <title>HTML Example</title>
</head>
<body>
    <p>This is an HTML document.</p>
</body>
</html>

XHTML Example


<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title>XHTML Example</title>
</head>
<body>
    <p>This is an XHTML document.</p>
    <br />
</body>
</html>
