HTML Image Maps

Creating Image Maps

An image map allows you to define clickable areas on an image. Use the <map> and <area> tags to create an image map. 


<img src="example.jpg" alt="Example Image" usemap="#examplemap">
<map name="examplemap">
    <area shape="rect" coords="34,44,270,350" href="https://www.example.com" alt="Example Area">
</map>
