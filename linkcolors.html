Styling Link Colors

You can style links using CSS to change their colors for different states:

  • a:link – Normal, unvisited link.
  • a:visited – Visited link.
  • a:hover – Link when hovered over.
  • a:active – Link when clicked.

a:link {
    color: #007BFF; /* Normal link */
}
a:visited {
    color: #800080; /* Visited link */
}
a:hover {
    color: #FF5733; /* Hovered link */
}
a:active {
    color: #00FF00; /* Active link */
}
