HTML Ordered Lists

Ordered List Styles

You can change the numbering style of an ordered list using the list-style-type property. 


<ol class="decimal">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
<ol class="upper-roman">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
<ol class="lower-alpha">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
